# init_nrf52_dev


## 目标
在win10中搭建nrf52的开发环境

## 方法


### 在win10中启动wsl2
略


### 安装编译环境 

````
git clone https://github.com/fugreat/init_nrf52_dev
cd init_nrf52_dev
sudo bash init_nrf52tools.sh
````



### 在VSCODE中安装插件

[Remote-WSL插件](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-wsl)

2. 在线调试工具
